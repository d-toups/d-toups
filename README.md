### 🚀 Featured Projects

#### [Workout Data Analysis](https://github.com/d-toups/workout-data-analysis)
Exploratory analysis of workout preferences and training intensity (calories/min) by gender and age group. Includes statistical hypothesis testing (Chi-square, t-tests) and visualizations with Seaborn.  
**Key Insight**: Age is a stronger driver of training intensity than gender.

#### [Stock Price Trend Prediction](https://github.com/d-toups/stock-price-trend-prediction)
Machine learning models (Random Forest vs. Gaussian Naive Bayes) to predict short-term stock movements using engineered features (skewness & relative volume). Uses proper TimeSeriesSplit cross-validation.  
**Best Result**: ~74.4% mean accuracy on selected configurations.

#### [Bookstore Database](https://github.com/d-toups/bookstore-database-sql)
Normalized relational database schema with analytical SQL queries, ERD, and data dictionary.
